전기차 충전 인프라 통합관제 웹페이지 및 API 백엔드 개발
2023.05 - 2023.06  (Team Project)
1/1
본 제품은 충전기를 이용한 충전 내역 및 관련 데이터를 수집하고 모니터링하는 프로그램입니다.

이 프로젝트는 전자정부 프레임워크(eGovFrame 3.9)를 기반으로 Java 1.8 및 Spring Boot를 사용하여 데이터 전송 최적화 및 실시간 모니터링 기능 API를 개발했습니다. 또한, Redis를 활용하여 실시간 데이터 저장 및 관리를 구현하였습니다.

본 서버는 Naver Cloud 플랫폼을 사용하여 운영되며, 데이터베이스는 MSSQL을 활용하고 있습니다. 이 경험을 통해 웹 개발에 대한 이해와 역량을 더욱 강화할 수 있었습니다.

주요 기능
  • 정보 등록 및 관리를 통한 운영 지원
  • 충전기의 현재 상태 모니터링 및 충전기에 대한 제어, 설정 등을 통한 충전 제어 명령
  • 충전 이력에 대한 통계 제공
  • 충전기의 고장 관리

    GS인증 1등급 인증 시험 합격
    2023.08 ~ 2024.01 
    1/8
    본 프로젝트는 GS인증 1등급을 획득하기 위해 시스템 아키텍처 설계 및 문서화 작업을 진행한 충전 인프라 프로젝트입니다.

    시험은 TTA에서 진행되었으며, ISO/IEC 25023, 25041, 25051 국제표준을 기반으로 수행되었습니다.

    인증 통과 시 시험성적서 및 품질인증서를 제공합니다.

    인증 제도 효과
    - 품질 개선 및 비용 절감
    - 고객 신뢰도 확보 및 제품 홍보
    - 조달청과의 수의계약 지원
    환경부/서울시 연계 API
     
    1/8
    본 프로젝트는 환경부와 서울시 전기차 충전 인프라 연계 API 개발입니다.

    충전소 위치 정보, 충전기 상태 정보, 회원 카드 인증, 충전 이력 관리 등을 위한 정보 공유를 목적으로 합니다.

    연계 방식은 HTTP 기반의 API 호출로 이루어지며, Restful API 방식으로 구현되었습니다. 데이터는 JSON 형식으로 제공되며, 보안 방식으로는 VPN 연결이 필요합니다.

    주요 기능
    - 충전소 정보 조회
    - 충전기 정보 관리
    - 회원 카드 관리
    - 충전 이력 관리
    - 이용 정보 관리

    컨넥티드 차량 관제
      (Team Project)
    1/8
    본 프로젝트는 IoT 기술을 활용한 차량 관제 웹페이지 및 API 백엔드 개발입니다.

    DTG 단말기를 통해 차량 센싱 정보를 NoSQL InfluxDB로 처리하며, MQTT 통신을 통해 어라운드뷰 및 공기청정기 센싱 정보를 실시간으로 전송하는 시스템을 구축했습니다.

    마이크로서비스 아키텍처를 기반으로 Docker를 사용하여 서비스 관리를 하였으며, 서비스 디스커버리 및 부하 분산을 위해 Eureka와 Spring Cloud Gateway를 사용했습니다.

    Keycloak을 이용하여 사용자 인증 및 권한 관리를 통한 보안 처리를 구현하였고, 데이터 시각화를 위해 Vue.js를 활용한 웹 인터페이스를 개발했습니다.

    Frontend
    Vue.js 3
    Backend
    Java 17, Spring Boot 3.2.2
    DataBase
    MSSQL, InfluxDB
