개인상점을 간편하게 관리하고 거래할 수 있도록 도와주는 중고거래 플랫폼 웹사이트입니다. 코로나19로 인한 여파로 소극적이게 변한 외부활동, 보다 더 편리하고 새로운 서비스로 사람들의 관심을 사 중고거래를 다시 활성화시키고자 하는 의도를 담아 개발했습니다.

이 프로젝트는 저의 첫 API 사용 경험이었습니다. API를 활용함으로써, 웹 개발에서 정보의 중요성과 데이터 연결의 복잡성을 체감할 수 있었고, API를 통해 외부 시스템과의 연동이 얼마나 강력한 도구가 될 수 있는지를 체험할 수 있었습니다.

최근에 인기가 많은 React 를 적용하여 각 컴포넌트가 독립적으로 동작하고 재사용 하여 가독성을 향상시키고 중복을 줄여주었습니다. 또한, Vritual DOM 을 이해하고 사용하면서 화면 렌더링 과정의 효율성에 대해 배울 수 있었고, React의 생명주기 메서드와 Hook 등의 개념을 이해하고 적용하면서, 동적인 웹 애플리케이션 개발에 대한 깊은 이해를 얻을 수 있었습니다.

이 경험은 저에게 많은 도전과 학습의 기회를 제공했고, 이를 통해 웹 개발에 대한 보다 깊은 이해와 역량을 키울 수 있었습니다. 앞으로도 React와 같은 최신 프레임워크와 API를 적극 활용하여, 효과적이고 최신 기술 기반의 프로젝트를 구축하겠습니다.

주요 기능
사용자 : 중고물품 판매, 급해요 시스템, 시세 조회, 잔고 충전 출금, 포인트 서비스, 이벤트(출석,룰렛), 채팅, 지역채팅, 고객문의, 광고문의
관리자 : 대시보드, 사용자 관리, 컨텐츠 관리, 마케팅 관리, 수입 관리, 통계, 메인페이지 관리, 고객지원

조직 간의 원활한 의사소통 및 효율적인 협업을 위한 다양한 비즈니스 기능을 지원하는 웹 페이지입니다. 대학생활을 하면서 교직원 분들이 사용하던 업무용 소프트웨어를 보고 아이디어를 얻어 개발하였습니다.

팀 프로젝트를 진행하면서 협업 하는 과정 에 관해 많이 배울 수 있었습니다. 요구사항 분석, Use Case 명세서, Use Case Diagram, Ckass Diagram, ERD Diagram, UI 와 같은 필요한 명세를 작성하며 소통하였고, 필요한 지식도 함께 공유하며 학습할 수 있었습니다.

클라이언트의 요청을 처리하고 그 결과를 반환하는 서버의 컴포넌트인 Jaca Servlet 을 통해 동적안 웹페이지를 구성하였습니다. 또한, Model, View, Controller 즉, MVC2 패턴을 적용하여, 웹 애플리케이션의 구조가 명확해지고, 유지보수 및 확장성이 용이해지는 장점을 경험하여 웹 개발에 대한 전반적인 이해도를 높이는 데 큰 도움이 된 프로젝트 경험이었습니다.

주요 기능
출결시스템, 근태현황 조회, 전자결재 시스템, 조직도 조회, 프로필 관리, 계정 생성, 계정 대량생성 [양식에 맞는 파일 등록],유저 생성

상품을 등록하고 구매할 수 있는 쇼핑몰 웹사이트입니다. Ajax와 jQuery, 그리고 Spring Boot Framework 를 적용해보기 위해 개발하였습니다.

웹 페이지를 리로드하지 않고도 서버와 데이터를 주고받을 수 있는 Ajax 를 적용하니 사용자 경험을 향상시키는 데 중요한 역할을 했습니다.

Spring Boot 를 처음 사용해 보았는데, 필요한 환경 설정을 대부분 자동화하여 비즈니스 로직에만 집중할 수 있었습니다. 이를 통해 RESTfull API를 빠르게 개발하고, JDBC를 이용한 데이터 베이스 연결 및 CRUD 작업을 수행하여 편리했습니다.

DataBase 와 연결하여 백엔드 개발을 진행한 첫 프로젝트 이며, 이 과정에서, 클라이언트와 서버 간의 데이터 교환, 데이터베이스 설계 및 관리, 서버 로직 구현 등 다양한 백엔드 개발의 요소를 직접 학습하고 적용해 볼 수 있었습니다.

데이터베이스 스키마 설계를 통해, 어떤 데이터를 어떻게 저장하고 관리할 것인지 결정하고, 스키마에 기반한 SQL 쿼리를 작성하고 실행하여 실제 데이터베이스틑 조작하고 관리하는 경험을 통해 어떤 서비스라도 만들 수 있을 것 같다는 자신감을 얻었습니다.

주요 기능
로그인, 회원가입, 상품등록, 상품 수정, 장바구니 담기, 상품 구매하기, 구매이력 조회
